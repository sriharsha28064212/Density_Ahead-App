# Density_Ahead-App
Clone the project

```bash
  git clone https://github.com/sriharsha28064212/Density_Ahead-App.git
```

Go to the project directory

```bash
  cd project-loc
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
